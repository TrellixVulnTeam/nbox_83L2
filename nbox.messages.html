nbox.messages.get(msg_or_dict, key, default=<object object>)[source]¶

Retrieve a key’s value from a protobuf Message or dictionary. :param mdg_or_dict: the

object.

Parameters

  • key (str) – The key to retrieve from the object.

  • default (Any) – If the key is not present on the object, and a default is set, returns that default instead. A type-appropriate falsy default is generally recommended, as protobuf messages almost always have default values for unset values and it is not always possible to tell the difference between a falsy value and an unset one. If no default is set then KeyError will be raised if the key is not present in the object.

Returns

The return value from the underlying Message or dict.

Return type

Any

Raises

  • KeyError – If the key is not found. Note that, for unset values, messages and dictionaries may not have consistent behavior.

  • TypeError – If msg_or_dict is not a Message or Mapping.

nbox.messages.setdefault(msg_or_dict, key, value)[source]¶

Set the key on a protobuf Message or dictionary to a given value if the current value is falsy. Because protobuf Messages do not distinguish between unset values and falsy ones particularly well (by design), this method treats any falsy value (e.g. 0, empty list) as a target to be overwritten, on both Messages and dictionaries. :param msg_or_dict: the

object.

Parameters

  • key (str) – The key on the object in question.

  • value (Any) – The value to set.

Raises

TypeError – If msg_or_dict is not a Message or dictionary.
